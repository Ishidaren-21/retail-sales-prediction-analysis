# 🛍️ Retail Sales Prediction and Business Insights Analysis

## 📖 Project Overview

This project presents an end-to-end data analytics workflow using a real-world retail sales dataset. The objective is to analyze customer purchasing behavior, identify key sales trends, and develop a machine learning model capable of predicting the total sales amount.

The project combines data preprocessing, exploratory data analysis (EDA), predictive modeling, and business recommendations to demonstrate practical applications of data science in the retail domain.

---

## 🎯 Objectives

- Understand the structure of the retail sales dataset.
- Perform data quality assessment and preprocessing.
- Explore customer purchasing behavior using EDA.
- Identify relationships between customer demographics, products, and sales.
- Build a Linear Regression model for sales prediction.
- Evaluate model performance using regression metrics.
- Generate actionable business insights and recommendations.

---

## 📂 Dataset Information

The dataset contains **1,000 retail transactions** with customer demographics and purchase details.

### Features

| Feature | Description |
|----------|-------------|
| Transaction ID | Unique transaction identifier |
| Date | Date of purchase |
| Customer ID | Unique customer identifier |
| Gender | Customer gender |
| Age | Customer age |
| Product Category | Purchased product category |
| Quantity | Number of items purchased |
| Price per Unit | Price of one product |
| Total Amount | Total transaction amount (Target Variable) |

## 📈 Model Performance

|---------|-------|
